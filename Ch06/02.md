# انشاء وعرض الفروع git branches

لعرض الفروع الحالية لمشروع محدد نستخدم الامر: 
<div style="text-align:center">

 
<div dir="ltr">


`git branch`


</div>


</div>
لانشاء فرع مستقل عن الفرع الاساسي :
<div style="text-align:center">

 
<div dir="ltr">


`git branch <new-branch-name>`


</div>


</div>


</div>
